    Release 1.1.0 - 02/24/2016

    Release 1.1.0 - 02/24/2016

    Some Bug Fixes/Enhancements:

    This was a BIG release and we're so excited to show you what we've enhanced!  Let's keep it to just the facts and break it down section by section. 

    • Resources:
      • *NEW* Demographics report! We're most excited about this feature and you're going to want to check this out!  Here's a glimpse at how we're making it easy for you to have fast, visual cues on all your resources and the stats that matter to you. 
      • Geographic map feature added, you can now view all your resources clearly, on a map. 
      • In addition to these, we did some back end housekeeping to clean up our geocoding on our maps.
      • Insights graph added so you can plot two points across different projects and compare the results.
      • We have edited the sort order on our project filter as per the requests of our users.  The project filter now sorts alphabetically.
      • Default Delegate if non-person type
      • Fixed save search function.
    • Global Resources:
      • We've made sure that none of your private resources will show in the "Global Resource" Search
    • Profile:
        • Send Invitation to ZenPRM functionality.  You can now give someone a little nudge to join ZenPRM.
        • "Claim This Resource" functionality (for non-persons)
        • Fixed scaling on NPI field to scale according to monitor size
        • Added "My Projects" when looking at "My Profile" page
        • Audit screen additions: We added auditing for the following: To Do, Project Requirement, Project Answer,  Project Answer Value. 
        • You can now add "Delegates"!  Check out this quick video on how easy it is: 

      AddDelegate from Val Grigoriou on Vimeo.

    • Project:
      • Added "Send Project Email" to Participants functionality so you can now message those within your project that you need to get a hold of.
      • We disabled the placeholder dropdown in the "Questions" section when not applicable
      • Also, we fixed a bug that wasn't clearing questions after changing from a non-option type question to another
      • A small UX fix to make your experience better, in participant "Projects" help text within form fields will insert when applicable
      • We enhanced the "Export Project" feature and ensured it was including all participants when the "Include Entire Network" flag is checked. 
    • Settings Menu:
      • Increase width of menu for a better UX experience

    Don’t forget, we have our knowledge base pages set up HERE to help guide you through the software.  You can also email us at the Zen Support help desk.
