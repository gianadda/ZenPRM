    Release 1.0.3.0 - 12/01/2015

    Bug Fixes:

    • Project screen

      • Resolved issue where the Toggle Switch Question would not save NO after a YES was selected

      • Resolved issue where the closed option would not save on the Hours of Operation question

    • Project - Questions

      • Resolved issue where a non-saved question was not accessible if the user clicked away and on to a different question

      • Resolved issue that would cause an error on deleting a question

    • Project – Export

      • Resolved a validation issue for the export column names that would prevent the export file from being generated

    • General

      • Resolved the alpha-sort of the delegates in an users sidebar menu

      • Resolved the session timeout issue that would not redirect user back to login when their session expired
