    Adding participants to a Project

    In ZenPRM, we give you all the tools you need to get your work done quickly and efficiently. As a result, we've implemented 5 different ways to add Participants to a Project. 

    1) If you already have existing projects, you can use one as a template for a new project. Here's how: 

    - Navigate to My Projects and Click "Add Project" from the top right button. 

    -Add a name, date and project manager as you would with any new project, then select a project from the "Start From Exsiting Project" dropdown.

    -Then choose to either include both participants and questions, only participants or only questions. 

    2) Add individual participants from your project's dashboard. 

    -Once you have set up your project, click your participants tab and add participants using the participant search box in the top right. 

    3) Include your entire network from the initial project set up screen. 

    - When creating a project for the first time, you can also add your entire network by selecting the appropriate check box. 

    4) Import your participants from the "Bulk Actions" screen. 

    -Once you have set up the basic information in a project, such as the name of it, the due date and the project manager, you can then navigate to "Bulk Actions" to import template files with questions, participants and/or answer data." 

    Including participant NPIs will automatically add them to the project when running a bulk action upload. 

    5) And finally, you can add participants to a project directly from the resources screen.
