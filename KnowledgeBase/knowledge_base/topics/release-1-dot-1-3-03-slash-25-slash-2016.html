    Release 1.1.3 - 03/25/2016

    Here are our latest enhancements and bug fixes.

    Projects

    o  We fixed a glich in the type-ahead feature that was allowing uses to save a partial answer. 

    o  We've added a project audit screen where you can see your project history. Check it out! 

    o Not only does your project have an audit screen, you can now see your question history as well! 

    UI Enhancenments

    o  We made our logo our loading icon on several screens (Projects, Profile, Resource Projects, Connections, Connections, Delegates, Interactions). The usual loading icons just weren't pretty enough for us. 

    o We also made some full screen loading UI enhancemnts as well as some to our dashboard.

    Activities

    o  We've made a lot of changes to the activities screen under the resources tabs.  We won't ruin the surprise here, go check it out yourself! 

    Resources

    o  We removed the Auto-search option from this screen.

    o  In addition, we fixed the "Is On Project" bug if "Include Entire Network" is checked

    o  Some changes were made to enhanced the performances of the "Export to CSV" function.

    o  And finally, we fixed a bug where a number wasn't displaying when there were no results

    Search

    o Finally, we updated the way our search functions work to give you the best exeperience we can offer.
