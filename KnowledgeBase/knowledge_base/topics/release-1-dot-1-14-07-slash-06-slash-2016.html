    Release 1.1.14 - 07/06/2016

    We hope everyone enjoyed a fun, safe and happy holiday weekend! We are back with some updates to ZenPRM to make your experience as simple and smooth as possible. Here's what you can expect from this release. 

    1. Improved the sorting and Drag & Drop functions of our Project questions. Here is what you can expect: 
      1. Increased drag handle size
      2. When the Add Question button is pressed, the question will be added immediately after the selected question (rather than at the bottom)
      3. Added keyboard shortcuts for sorting to top, up, down, and to bottom
      4. When dragging a question, the panel will now scroll when you reach the bottom or top (depending on drag direction)
      5. On mobile, added a row of buttons below the selected question to sort to top, up, down, or to bottom.
      6. Added toggle arrow on the right side of each question so the edit panel can be closed or opened manually
      7. Added a close icon within the edit panel so it can be closed manually
      8. Double-clicking on question row will toggle edit panel display
      9. Added keyboard shortcut to delete question with Delete key.
    2. Project - Import Bug Fixes
      1. Fixed bug that prevented adding participants and data on initial import (if adding new questions)
    3. Question Drill down - Download file
      1. If you have a project where your participants provided you with a file, the Question Drill Down screens will allow you to download that file from that location as well. 
    4. Project Questions - 'Add Question' bug fix. 
      1. Previously, a user was able to click 'Add Question' before saving the previous question, which would then lose the previous unsaved data.  This is no longer the case.  You cannot add a new question until you have saved. 
    5. Edit Profile - Update templates per resource
      1. We've made some updates to our resource profiles. 
        1. General Contacts no longer have the 'Additional Information' 'NPI' and the 'Licenses and Other IDs' fields. 
        2. Committees no longer have the 'NPI', 'Taxonomies', 'Services' and 'Systems' fields. 
        3. Other resources no longer have the 'NPI', 'Taxonomies', 'Services' and 'Systems' fields.  
    6. Resource Directive:  Profile photo displaying
      1. We fixed a bug that was not allowing profile photos to show on the Question Drill Down page, the My Resources Page, the Project Audit and Project Participants screens. You now have a profile thumnail on all of those screens when available. 
    7. Resource Profile -
      1.  We fixed a bug where 'Bookmarked Projects' were not appearing for Organizations.
