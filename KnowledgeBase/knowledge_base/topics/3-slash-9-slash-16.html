    Release 1.1.2 - 3/8/2016

     March 8th, 2016

    Here are our latest enhancements and bug fixes.

    Projects

    o  There was a small bug that was preventing the first question from being added to a new project. We fixed that.

    o  Check out the new modal that allows you to flip through you participants questions quickly and easily.

    o  We fixed a glitch that allowed any "hours of operations" questions as complete even if they had not been filled out.

    o  We also updated our save message on the project screen to reflect some more accurate information

    o  You can now clear way answers in projects, this will come in handy in a lot of instances, like when you need to copy the same project over monthly. Speaking of which....

    o  You know have the ability to copy an existing project and use it over and over again, pretty neat huh? How many of you have recurring projects you need to track?

    Navigation 

    o  If you select one of the navigation items from the left hand menu, they will stay highlighted and selected through all the sub/child screens.

    Registration

    o  We fixed the bug causing issues when trying to register delegates

    Resources

    o  You can now filter your resources by the option of "Is on Project"

    o  We have updated the project filter sort order on drop down.

    o  Updated distance to just one decimal place.

    Demographics

    o  Added some "quick links" to the demographics graphs to help you better understand the data.  Check out the Video: 

    Demographics_tooltips from ZenPRM on Vimeo.

    o  Added tool tip to "view report" icon 

    o  Make sure view report link is not added to “N/A” values            

    o  Improved loading display.

    Map

    o  There is now an “Unmappable Location" quick link as a search filter.

    As always, you can email our support team at zensupport@ahealthech.com if you have any questions or concerns .
