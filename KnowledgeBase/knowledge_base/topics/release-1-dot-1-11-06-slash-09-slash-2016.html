    Release 1.1.11- 06/09/2016

     

    Welcome back ZenPRM users! With a little break over the holiday weekend and some extra time spent dialing in some of our exisiting features this week, we've become quite excited because we made ZenPRM even more amazingly simple!

    Here is what we have done for you.  

     

    First, the behind the scenes magic:

    - The system geo-coding now prioritizes your network  

    - We fixed a validation issue on the 'Address2' field under profiles 

     

    And some bug fixes: 

    -We fixed a bug that that was including the wrong 'closed' date on 'Tickets'

    -Also fixed, was a bug that was cutting off text when a project title was too long. 

    -the 'Help' menu is now fully functional in mobile devices as well. Accessing the menu was previously a little difficult.

    -We fixed a bug where 'cleared' answers to projects were not auditing. All that information is now being saved in the 'Audit' tab. 

     

    And some general enhancements:  

    - You now have the ability to edit 'Notes' under a resource's profile. 

     

    -We've cleared up our registration screen to make the registration process as simple as possible. 

    -You can now create 'Key Performance Measures' with non-numeric answer types such as Yes/No. 

    -We've improved the experience when adding a project, there is no more redirecting off of that page after you 'Save'. 

    -We've spent quite a bit of time making sure the date pickers are consistent across all major modern browsers. 

    -We have added some restrictions to what file types can be uploaded.  See the full list here. 

    -There is now a 'Send Registration' button on resource profile screens for you to quickly send folks access to ZenPRM. 

     

    As always, please do not hesitate to reach out to us at zensupport@ahealthtech.com should you have any questions.
