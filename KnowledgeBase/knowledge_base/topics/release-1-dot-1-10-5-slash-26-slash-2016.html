    Release 1.1.10 - 5/26/2016

    Greetings ZenPRM Clients! 

    We have been working on quite a few more fine details in ZenPRM once again with the goal of maximizing the ease and enjoyment of your user experience. Here is what we are releasing this week. 

    1) Guides.

         a) Throughout our system we have initiated help guides to walk you through. You may have noticed them the first time you logged in. You'll see more of these as we add new features we'd like to highlight.  We have also added a static 'Help' menu where you can search for more information about the page you are working on. 

     

    2) ZenPRM template projects. 

         a) We've uploaded some example projects into ZenPRM.  First time users will see these projects on their dashboard. They are a tool for you to modify and resuse, delete or archive as you see fit. Its our way of giving you a glimpse into what ZenPRM can do. 

    3) Introducing a WHYSIWYG for all your formatting needs! We've added some features so you can format your text under the question descriptions. 

    4) Reactivate Projects.  We have added a button to reactivate archived projects.  

    5) Reformatted the 'Hours of Operation' display to make it more visually appealing.

    6) User emails are no longer editable by anyone other than the verified user of that account. 

    7) Bug Fixes: 

         a) KPMs: Fixed a bug that would load previously edited dial when clicking new measure

         b) Previously, one was able to save a question without type, the save succeeded and question disappeared.  Now you can no longer save a question without a type. 

         c) Archived project questions can no longer be updated by the participant. 

         d) We corrected and issue that was preventing projects from appearing in the list after a bulk add, we also removed archived projects from the bulk add.
