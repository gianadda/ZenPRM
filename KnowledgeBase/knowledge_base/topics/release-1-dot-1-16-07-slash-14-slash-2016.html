    Release 1.1.16 - 07/21/2016

    First, lets start with a couple of the small updates we made to ZenPRM this week. 

    • We fixed a bug when adding a second question to a project. Now, when you add another quesion, it will focus right on that question. 
    • We've implemented options for bulk adding questions.

    And now, what we're really excited for! The team at ZenPRM is truly committed to providing you with the best tools and resources we can to help you better manage your provider network. From the beginning, we set out to create a tool that not only kept you organized, but also gave you valuable data from the very first time you logged in. The first inception was pulling in provider demographic information sourced from public data sets such as the NPPES directory.  This evolution of ZenPRM takes that one giant leap forward. We have spent some time mining and refining public data from CMS to provide you with Medicare data and referrals. All of this valuable information is now located right on your Resources profile.
