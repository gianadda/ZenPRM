    Release 1.1.5- 04/21/2016

    Another week another set of enhacements from the team at ZenPRM

    1)    First things first, some UI enhancements.

    We've made some adjustments to our panels to make them cleaner and more visually appealing. We hope you like them! 

    2)    NEW Feature! Bulk Upload Project Data

    You can now upload a file to create a project and add questions or update an existing project! How neat is that? 

    3)    API change to improve performance

    We did some backend improvements to help with performance.  We changed the session handling to only exclusive lock on login/account maintenance (writes) which should improve asynchronous capabilities. 

    4)    New Feature: Bulk Add Users

    In addition to the bulk projects and questions, you can now bulk add users too! 

    5) And now for our favorite part of this release....drum roll please.... Key Performance Measures for your projects! 

    Take a look at the video below as we walk you through this exciting new feature. Keep an eye on the data that matters to you, when it matters to you, right on your dashboard!
