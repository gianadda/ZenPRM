    Project

    A project is simply a collection of data that pertains to one or more Resources. A project may be used to store information that extends Resource profiles, gather data for reporting purposes, or survey members of your network. An archived project is one that is no longer actively being worked on, but you still want a record of.  Below is the project dashboard which highlights KPMs and shows your project question and percent completion.
